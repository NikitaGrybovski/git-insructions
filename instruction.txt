commands git
1.создание локального репозитория
	git init
2.Проверка состояния репозитория
	git status
3.Добавление файлов в репозиторий
	git add
4.Удаление ошибочно добавленных файлов из под отслеживания
	git reset HEAD ...
5.Создание файла gitignore
	touch .gitignore
6.Запись в .gitignore игнорированных файлов и дирикторий/
	echo "filename" >> .gitignore
7.Проверка состояния репозитория и добавление изменений. 
	git status
	git add
8. Создание слепка файлов в репозиторий. Перед этим проверить чтобы все файлы были зеленые.
	git commit -m ...
9. Создать репазиторий на github.com с именем локального репозитория
10.Связыванние удаленного и локального репозитория github.com/<youlogin>/<youreponame.git>
11.Как узнать с каким удаленным репозиторием связанн локальный
	git remove -v
12.Как удалить связь с удаленным репозиторием?
	git remote -rm origin
13. Отправка файлов из локального репозитория на удаленный
	git push -m origin master
14.Как вытянуть файлы из удаленного репозитория
	git pull link
